# tinder

## Инициализация проекта

#### Инициализация клиента
```
yarn install
```
#### Инициализация сервера
```
cd server
```

```
yarn install
```
## Запуск проекта

#### Компиляция клиента с горячей-перезагрузкой для разработки
```
yarn serve
```

#### Компиляция сервера с горячей-перезагрузкой для разработки
```
cd server
```

```
yarn server
```

#### Или запуск сервера
```
cd server
```

```
yarn start
```
